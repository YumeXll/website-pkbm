# PKBM Bale Rumawat - Website

Website resmi PKBM Bale Rumawat - Jakarta Timur

## Struktur Folder

```
website-pkbm/
│
├── index.html                 # Halaman utama
├── css/
│   └── style.css             # File CSS utama
├── js/
│   └── main.js               # File JavaScript utama
├── images/                   # Folder untuk gambar
│   ├── logo.png
│   ├── guru-alt-PKBM.png
│   ├── elements/             # Elemen dekoratif
│   ├── partners/             # Logo partner
│   ├── testimonials/         # Foto testimonial
│   └── blog/                 # Gambar blog
└── pages/                    # Halaman-halaman lain
    ├── sejarah.html
    ├── visi-misi.html
```

## Fitur

- Responsive design untuk semua ukuran layar
- Menu navigasi dengan dropdown
- Search functionality
- Mobile-friendly navigation
- Smooth scrolling
- Animation on scroll
- Social media integration
- WhatsApp integration
- SEO friendly

## Teknologi yang Digunakan

- HTML5
- CSS3 (Modern CSS dengan CSS Grid & Flexbox)
- JavaScript (Vanilla JS)
- Font Awesome Icons
- Google Fonts (optional)

## Cara Menggunakan

1. Extract semua file ke folder web server Anda
2. Pastikan struktur folder sesuai dengan yang tertera di atas
3. Tambahkan gambar-gambar yang diperlukan ke folder `images/`
4. Buka `index.html` di browser untuk melihat hasilnya

## Kustomisasi

### Warna
Ubah warna di file `css/style.css` pada bagian `:root`:

```css
:root {
    --primary-color: #16a34a;
    --secondary-color: #84cc16;
    --dark-color: #14532d;
    --light-color: #f0fdf4;
    --text-color: #334155;
}
```

### Konten
Edit konten di file `index.html` atau file HTML lainnya sesuai kebutuhan.

### Menu
Edit menu navigasi di bagian `<nav class="main-nav">` di file `index.html`.
